# Seguridad

Puente Impacto puede trabajar con informaci&oacute;n relacionada con organizaciones, escuelas, estudiantes, programas sociales y proyectos comunitarios.

La seguridad y privacidad deben tratarse con cuidado desde el dise&ntilde;o.

## Reportar un problema

Si identificas una vulnerabilidad o problema de seguridad, evita abrir un issue p&uacute;blico con detalles sensibles.

Contacto:

- contacto@puenteimpacto.org

## Datos sensibles

No deben subirse al repositorio:

- Archivos `.env`.
- Tokens.
- Llaves privadas.
- Credenciales.
- Datos personales reales.
- Bases de datos exportadas.
- Documentos internos de clientes.
- Informaci&oacute;n confidencial de organizaciones.

## Buenas pr&aacute;cticas

- Usar variables de entorno para secretos.
- Revisar permisos antes de integrar servicios externos.
- Minimizar datos personales recolectados.
- Documentar supuestos y responsabilidades.
- Validar formularios y rutas sensibles.

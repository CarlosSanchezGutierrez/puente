# Puente Vocacional 2026

Programa de orientacion vocacional para preparatorias durante el semestre Agosto-Diciembre 2026.

## Objetivo

Conectar estudiantes de preparatoria con carreras, universidades y profesiones mediante conversaciones con universitarios, practicantes, profesionistas, profesores, directivos y orientadores vocacionales.

## Metricas iniciales

- Preparatorias visitadas
- Estudiantes alcanzados
- Areas vocacionales representadas
- Perfiles participantes
- Conexiones vocacionales
- Recursos compartidos

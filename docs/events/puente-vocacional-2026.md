# Puente Vocacional 2026

Programa de orientacion vocacional para preparatorias durante el semestre Agosto-Diciembre 2026.

## Objetivo

Conectar estudiantes de preparatoria con carreras, universidades y profesiones mediante conversaciones honestas con universitarios, practicantes, profesionistas, profesores, directivos y orientadores vocacionales.

## Enfoque

No imponer una carrera ni vender una universidad. Facilitar informacion real, contactos y recursos para que los estudiantes tomen decisiones mas informadas.

## Metricas iniciales

- Preparatorias visitadas
- Estudiantes alcanzados
- Areas vocacionales representadas
- Mentores y perfiles participantes
- Conexiones vocacionales
- Recursos compartidos

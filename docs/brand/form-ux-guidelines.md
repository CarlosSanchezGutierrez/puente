# Form UX Guidelines

Puente forms should feel:

- simple
- calm
- serious
- mobile-friendly
- low-friction
- trustworthy

## Rules

- Ask only for information needed to follow up.
- Keep WhatsApp or email follow-up as the serious channel.
- Do not make public forms feel like enterprise procurement.
- Avoid unnecessary required fields.
- Make consent and privacy visible but not intimidating.
- Use clear labels instead of vague placeholders.

## Current public forms

- Contact message
- NGO request
- Volunteer application
- Book request
- Event registration

# Visual Direction

Puente should feel:

- sober
- human
- institutional
- accessible
- technical without looking corporate
- social without sounding performative

## Typography

Current direction:

- Instrument Sans for interface and body text
- Source Serif 4 for high-level editorial headlines

## Visual tone

The site should avoid startup clichés.

Preferred feeling:

public institution + serious civic technology + community project + research lab
UI principles

soft borders

# Puente Impacto

**Tecnolog&iacute;a social para proyectos reales.**

## Prop&oacute;sito

Puente Impacto busca crear puentes entre conocimiento, comunidad y acci&oacute;n.

La idea es que estudiantes, organizaciones, escuelas, mentores, profesionistas y aliados puedan colaborar en proyectos reales que dejen utilidad concreta: herramientas, evidencia, aprendizaje, documentaci&oacute;n, seguimiento y soluciones que puedan mantenerse y mejorar con el tiempo.

El enfoque no es construir tecnolog&iacute;a por moda. El enfoque es entender una necesidad, definir un alcance claro, documentar el proceso y construir soluciones &uacute;tiles.

## Desarrollo local

Requisitos generales:

- Node.js
- pnpm
- Git

Instalaci&oacute;n:

```bash
pnpm install
```

Ejecutar sitio web:

```bash
pnpm --filter web dev
```

Validar calidad:

```bash
pnpm --filter web lint
pnpm --filter web build
```
